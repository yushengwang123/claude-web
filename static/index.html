    使用技巧
    • 粘贴图片直接发送（支持截图）
    • 粘贴 URL 自动抓取网页正文作为上下文
    • 拖拽文档（PDF / DOCX / XLSX 等）上传并提取文本
    • 点击输入框工具栏的魔法棒，基于黄金样本和个人规则优化当前 Prompt
    • Chrome 插件支持网页选中即问，结果可在 Side Panel 流式查看并转入 Web 会话
    • 给回答点 👍 或收藏后，可作为 Prompt 优化器的黄金样本候选
    • 工具调用失败时可一键放行权限重试
    • Git 仓库自动 checkpoint，可一键回滚到对话前
    • 双击会话标题重命名，或点 🪄 让 AI 命名
